# Relation V2 Ten-Subject Acquisition Gate

This is the compact, versioned training/evaluation package for the first M1 Relation V2 run.
It contains the 5,000-row V2 canonical profile table required to construct candidate inventories
and the complete 10-subject / 50-fact direct-aware gate. The 90 MB full generated corpus remains
reproducible in `synthetic-data-generation` and is intentionally not duplicated here.

The source release and every tracked file hash are pinned in `manifest.json`.
